# 사전조사 (Research)

GPS 없이 스마트폰 센서만으로 실내 위치를 추적하고 길찾기를 제공하는 시스템을 만들기 위한 사전조사 모음이다.
야외에서 실내로 진입하면 자동으로 실내 지도로 전환되는 경험을 목표로 한다.

## 한 줄 정의

> **추가 인프라(비콘·Wi-Fi 측위맵·VPS 서버) 없이, 스마트폰 IMU 센서 + 건물 평면도만으로
> 동작하는 Map-Aided PDR 기반 실내 내비게이션.**

## 핵심 결론 요약

1. **PDR 단독은 누적 오차가 치명적**이다(이동거리의 5~15%). 하지만 평면도 제약(Particle Filter)을
   결합하면 복도가 많은 건물에서 **1~3m 수준**까지 줄어든다. → 03 문서.
2. **방향(heading)이 가장 약한 고리**다. 자이로는 드리프트, 지자기는 자기 간섭. 둘을 융합한 AHRS가
   필수이며, 평면도가 방향 오차도 함께 보정해준다. → 02·03 문서.
3. **자이로가 없는 저가 안드로이드는 사실상 타겟에서 제외**. 최소 사양을
   `가속도계 + 자이로 + 지자기`로 못박는 게 현실적이다. → 05 문서.
4. **자동 전환은 GPS 입구 좌표 매칭 + 정확도(accuracy) 급락 감지** 조합이 가장 구현 쉽고 안정적.
   → 04 문서.
5. **경진대회 차별점은 "인프라 0"**이다. 비콘·VPS 없이 평면도 한 장으로 동작한다는 점을 전면에 둔다.
   → 07 문서.
6. **RAG로 자연어 UX를 추가**하면 두 번째 차별점이 생긴다. 자연어 목적지 파싱 → 건물 정보 Q&A →
   실시간 경로 우회 순서로 구현 우선순위를 잡는다. → 09 문서.
