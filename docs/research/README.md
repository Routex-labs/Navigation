# 사전조사

현재 `research/`는 구현 방향을 정하는 데 필요한 최소 근거만 남긴다.

이 프로젝트의 현재 방향은 **Flutter 클라이언트 + FastAPI 백엔드 + SQLite 기반 실내 지도/경로 API**다.
초기 조사에 있던 PDR, Particle Filter, 센서 융합, 자동 실내/외 전환, RAG 구현 문서는 아직 실제
구현 범위가 아니므로 제거했다.

## 정리 기준

- 현재 코드와 실행 방법에 직접 연결되는 내용만 남긴다.
- 구현되지 않은 기능은 기술 스택 문서의 "후속 후보" 수준으로만 기록한다.
