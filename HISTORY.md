## 2026-06-29 · 주제 확정 - 실내 PDR 내비게이션 + 사전조사

- 프로젝트 주제를 **GPS 없이 스마트폰 센서만으로 동작하는 Map-Aided PDR 기반 실내 내비게이션**으로 확정했다.
- 기존 `docs/research-notes.md`의 "보행 약자 접근성 경로 추천" 방향은 초기 자동화 생성물이었고,
  실제 진행 주제와 달라 실내 내비게이션 주제로 교체했다. (관련 논의로 방향 전환)
- `docs/research/`에 주제별 사전조사 9종(00~08)을 추가했다: 문제정의, PDR, 센서융합, 지도매칭,
  실내외 전환, 기기 호환성, 기술 스택, 경쟁 솔루션, 평가 방법.
- 후속 과제: `docs/navigation-overview.md`와 `VERSION.md`의 일반 웹 스택 가정을
  실제 스택(Flutter + FastAPI + Dart 알고리즘)으로 갱신 필요.

## 2026-06-29 · 기술 스택 상세화 + RAG 통합

- `docs/research/09-rag-integration.md`를 추가했다: 자연어 목적지 파싱, 건물 정보 Q&A,
  실시간 경로 우회, 다국어 안내 4가지 적용 지점과 FAISS 기반 경량 RAG 스택을 정리.
- `docs/research/06-tech-stack.md`를 대폭 상세화했다: 레이어 구조, 패키지별 목표 버전,
  클라이언트/백엔드 디렉토리 구조, RAG 레이어, 개발·배포 도구를 추가.
- `VERSION.md`의 컴포넌트 표를 실제 스택(Flutter/Dart 클라이언트 + 온디바이스 측위 엔진
  + FastAPI + RAG)으로 교체해 위 후속 과제를 해소했다.

## 2026-06-29 · v0.1.0 - 초기 메타 문서 정비

- 기존 운영 문서 구조를 Navigation 프로젝트용으로 재구성했다.
